1. Написать программу, считывающую массив строк (2 - 3 штуки) (до нажатия клавиши Enter) в сегмент данных. 
Размер строки не больше 240 символов. 
Сначала ввод всех строк, затем вывод всех строк.

2. Написать программу, считывающую строку (до нажатия клавиши Enter) в массив строк в сегменте данных в количестве трех штук.
 Размер строки не больше 240 символов. После вывести считанные строки, разделяя их a) пробелом, б) точкой с запятой, в) переводом строки.
 Все три варианта вывода реализовать в одной программе.

3. Написать программу, считывающую строку (до нажатия клавиши Enter) в сегмент данных. Размер строки не больше 240 символов. После вывести эту строку на экран. 
Использовать файловый сервис DOS (функции прерывания int 21h с номерами 0x3c-0x46).
Для считывания строки читать из стандартного ввода (дескриптор 0).
Для вывода строки писать в стандартный вывод (дескриптор 1).

4. Написать программу для вычисления формулы z = (x*y) / (x + y). 
Написать программу для вычисления формулы квадрата суммы ( с = (a + b)^2 ).
Написать программу для вычисления формулы куба суммы ( с = (a + b)^3 ). 
Все числа 16-битные целые со знаком.
